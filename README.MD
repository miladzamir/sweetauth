## üî®Installation:

### 1. Composer
```
composer require miladzamir/sweetauth
```

### 2. Add the ServiceProvider
Open the file config/app.php
```php
"providers": {
    ...
    MiladZamir\SweetAuth\SweetAuthServiceProvider::class,
    ...
},
```

### 3. Publish vendor resources
```
php artisan vendor:publish
```

### 4. Migrate tables
```
php artisan migrate
```
